API REST PARA CREACION DE USUARIOS
-----------------------------------------------------------------------------------

API desarrollada con Java 8, Spring Boot, Gradle, JPA y JWT.

La aplicación utiliza una base de datos en memoria (H2) para almacenar usuarios.
